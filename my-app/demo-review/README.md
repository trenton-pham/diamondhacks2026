# Demo Review Package

This folder is the approval package for the next demo dataset refresh.

It contains:
- `raw-texts/`: sample source text logs for Jordan and 6 candidate bots
- `chat-drafts/`: proposed one-to-one thread transcripts between Jordan and each bot
- `demo-cast.md`: compatibility tiers, posts, and rationale for each demo bot

Important constraints for this package:
- All chat drafts are `Jordan <-> bot` only.
- No bot-to-bot interaction is included.
- This content is not yet wired into the live backend seed data.
- After approval, the current noisy posts/chats should be wiped once and replaced with this curated dataset.
